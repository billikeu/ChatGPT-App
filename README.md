# billikeu/ChatGPT-App

Only 5.1MB, [ChatGPT-App](https://github.com/billikeu/ChatGPT-App) is a desktop chatbot application. Support multiple chat engines including `ChatGPT`, `New Bing`,`Bard`, `yiyan`, etc.

## Plan

- [x] ChatGPT
- [ ] ChatGPT Unofficial
- [x] Bing Unofficial
- [ ] Bard
- [ ] YiYan

## Support Platform

- [x] Windows
- [x] Linux
- [x] MacOs
